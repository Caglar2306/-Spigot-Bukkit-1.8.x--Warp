# [Spigot-Bukkit-1.8.x] Warp
The perfect warp plugin for your server!

# Commands
/setwarp warpname

/updatewarp warpname

/warp list|remove|warpname

# Features

    It's multilingual
    Clean code
    Support chat colors
    Visitor numbers are saved
